


# 泰坦尼克号乘客生存预测（Titanic Survival Prediction）

## 一、背景介绍

1912 年，泰坦尼克号在其首航途中不幸沉没，造成大量人员伤亡。  
历史数据表明，乘客的生存情况与其性别、年龄、舱位等级、是否携带家属等因素密切相关。

本题通过对历史乘客数据进行建模，预测乘客是否在事故中幸存，是一个典型的二分类问题。

---

## 二、任务描述

给定乘客的个人信息和船舱信息，请你训练一个分类模型，预测测试集中每位乘客是否幸存。

- 幸存：`1`
- 未幸存：`0`

---

## 三、数据集说明

### 3.1 数据来源（开源）

- Kaggle：Titanic - Machine Learning from Disaster  
  https://www.kaggle.com/competitions/titanic

---

### 3.2 数据文件

- `train.csv`：训练数据，包含特征和生存标签
- `test.csv`：测试数据，仅包含特征

---

### 3.3 字段说明（部分）

| 字段名 | 含义 |
|------|------|
| PassengerId | 乘客唯一编号 |
| Pclass | 舱位等级 |
| Sex | 性别 |
| Age | 年龄 |
| SibSp | 同船兄弟姐妹/配偶数量 |
| Parch | 同船父母/子女数量 |
| Fare | 船票价格 |
| Embarked | 登船港口 |
| Survived | 是否幸存（仅训练集提供） |

## 五、评分规则

本题首先计算预测结果在测试集上的 **准确率（Accuracy）**：

[
Accuracy = \frac{\text{预测正确的样本数}}{\text{总样本数}}
]

最终得分计算公式为：

[
Score = \min(100,; Accuracy \times 100 + 10)
]

说明：

* 准确率以区间 ([0,1]) 计算
* 最终分数上限为 **100 分**
* 排名按最终得分从高到低排序
